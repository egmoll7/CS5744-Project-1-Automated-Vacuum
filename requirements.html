Stair Climbing Vacuum
Background Use Cases Requirements High-Level Design Modules Design Rationale Conclusion

Requirements

Functional Requirements

FR-1. The system must retain the basic vacuum functions:
FR-1.1. Must be able to collect dust.
FR-1.2. Must be able to pick up pet hair.
FR-2. The system must be able to navigate the floorplan unassisted:
FR-2.1. The system must be able to detect when each floor is finished and move to the next.
FR-3. The system must be able to detect obstacles and change route accordingly.
FR-4. The system must be able to detect stairs vs other objects.
FR-5. The system must be able to climb stairs:
FR-5.1 Must be able to not fall off the stairs.
FR-6. The system must be able to go down the stairs.
FR-7. The system must be able to go to the docking station when vacuuming is done.
FR-8. The system must be able to integrate with a mobile application in order receive configuration and report status.
FR-9. The system must allow the configuration of floorplan bounds via the mobile app.
FR-10. The system needs to protect itself if stepped on:
FR-10.1. Must be able to continue if it is a safe weight.
FR-10.2. The system will stop moving and make a sound that will scare the person or pet off of the machine if an unsafe weight.
FR-11. The system needs to be able to alert the owner if the system is stuck somewhere.

Non-Functional Requirements

NFR-1. The system shall be able to operate for at least 2 hours on charge.
NFR-2. The system shall return to the docking station if the battery level is under 10 percent.
NFR-3. Once recharged the system shall resume the cleaning automatically if not previously completed:
NFR-3.1. If the system needs to recharge, it has to store which areas have been covered and which have not to continue when it went to charge.
NFR-4. The system shall indicate the battery level with an LED light:
NFR-4.1. If the battery is discharged the system shall display a solid red light.
NFR-4.2. If the battery is too low to start cleaning the system shall display a flashing red light.
NFR-4.3. If the systemâ€™s battery is charging the light indication shall display a flashing amber light.
NFR-4.4. If the system is fully charged the light indicator shall display a solid green light.
NFR-5. The system will clearly display when vacuum bags need to be changed.
NFR-6. When full the system will not run.
NFR-7. There needs to be an easy Vacuum bag changing system.
NFR-8. There should be a storage for bags on top of the charging station.
NFR-9. Status of the vacuum, i.e. not cleaned that day, cleaning, and cleaned.
NFR-10. Status of the charge level of the Vacuum.
